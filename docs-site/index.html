Quickstart

ApexPay to API do portfeli, rozliczeń marketplace (split na subkontach), płatności redirect (Autopay BM), wypłat i webhooków B2B. Bazowy URL produkcji: https://api.apexpay.pl — endpointy są dostępne pod /api/v1/....

Trzy minimalne kroki do pierwszej transakcji marketplace: wygeneruj klucz API, utwórz subkonto (Connected Account), zainicjuj charge ze splitem. Po prawej — gotowe wywołania curl.

Krok 1 — wygeneruj klucz API

Zaloguj się do panelu (JWT), a następnie utwórz klucz integracyjny. Pełny sekret apx_live_... zwracany jest tylko raz — zapisz go w menedżerze sekretów.

POST /api/v1/api-keys 
curl -sS -X POST "https://api.apexpay.pl/api/v1/api-keys" \
  -H "Authorization: Bearer YOUR_JWT" \
  -H "Content-Type: application/json" \
  -d '{"name":"Sandbox integrator"}'

Krok 2 — utwórz subkonto (Connected Account)

Onboarding KYC dla beneficjenta: body zawiera e-mail, typ (INDIVIDUAL / COMPANY) i kraj. Odpowiedź zawiera identyfikator subkonta — użyj go w splitach charge.

POST /api/v1/integrations/accounts 
curl -sS -X POST "https://api.apexpay.pl/api/v1/integrations/accounts" \
  -H "x-api-key: apx_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "seller@example.com",
    "type": "INDIVIDUAL",
    "country": "PL"
  }'

Krok 3 — zainicjuj charge ze splitem

Marketplace charge obciąża portfel integratora pełną kwotą i rozdziela środki na subkonta zgodnie z tablicą splits. Wymagany nagłówek Idempotency-Key.

POST /api/v1/integrations/charges 
curl -sS -X POST "https://api.apexpay.pl/api/v1/integrations/charges" \
  -H "x-api-key: apx_live_YOUR_KEY" \
  -H "Idempotency-Key: charge-demo-001" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 10000,
    "currency": "PLN",
    "splits": [
      { "connectedAccountId": "CONNECTED_ACCOUNT_UUID_A", "amount": 7000 },
      { "connectedAccountId": "CONNECTED_ACCOUNT_UUID_B", "amount": 2000 }
    ]
  }'
Dalej Skonfiguruj webhook URL i sekret w GET/PUT /api/v1/integrations/config, aby otrzymywać zdarzenia charge.succeeded.
